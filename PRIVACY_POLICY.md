# Privacy Policy

## WhatsApp Message Assistant – Chrome Extension

Effective Date: 15 Apr 2026

---

## 1. Overview

This extension helps users send personalized messages via WhatsApp Web in a controlled, user-initiated way.

---

## 2. Data Collection

This extension does NOT collect, store, or transmit any personal data externally.

- No tracking
- No analytics
- No third-party services

---

## 3. Data Usage

All data (contacts, messages) is processed locally in the browser only.

---

## 4. Data Storage

Temporary data is stored in chrome.storage.local and remains on the user's device.

---

## 5. Permissions

- storage → local settings
- scripting / activeTab / tabs → interact with WhatsApp Web
- host_permissions → <https://web.whatsapp.com/>*

---

## 6. Third-Party Sharing

No data is shared.

---

## 7. User Control

Users control all actions and can remove data anytime.

---

## 8. Disclaimer

Not affiliated with WhatsApp or Meta. Users must follow WhatsApp terms.
